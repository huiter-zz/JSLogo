命令讲解及DEMO

FD 100: 向前走100个单位

BK 50: 向后走50个单位

LT 30: 向左转30度

RT 90: 向右转90度

PU: 抬起笔，此时再执行移动的操作就不会留下痕迹

PD: 落下笔，此时移动会留下痕迹，这个为默认状态

REPEAT 4 [ FD 90 LT 90 ]: 重复4次【】中的操作

HOME: 回到初始位置

RESET: 还原到初始状态

COLOR RED: 设置画笔颜色为红色，也支持#000000这种写法

复杂点的DEMO

COLOR RED REPEAT 20 [FD 100 LT 150]: 12角星

FD 100 RT 90 COLOR RED REPEAT 20 [FD 40 RT 90 FD 1 RT 90 FD 40 LT 90 FD 1 LT 90]: 小红旗
